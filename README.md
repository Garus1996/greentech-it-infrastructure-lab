# GreenTech Solutions AS - Small Business IT Infrastructure Lab

## Project Overview

This project simulates a small business IT environment for a fictional company called GreenTech Solutions AS.

The goal of the lab is to demonstrate practical skills in IT infrastructure, system administration, automation and basic security hardening.

## Planned Lab Components

- Windows Server Active Directory
- User and group management
- Shared folder permissions
- Windows client joined to domain
- Ubuntu Linux server
- Docker Compose service stack
- Backup automation
- Basic logging
- Security hardening checklist
- GitHub Actions CI/CD basics
- Terraform infrastructure-as-code basics

## Skills Demonstrated

- Windows Server administration
- Active Directory
- DNS and domain services
- Linux server administration
- Docker and Docker Compose
- Git and GitHub
- Technical documentation
- Access control
- Backup and recovery
- Security hardening

## Lab Architecture

The lab will include:

- 1 Windows Server VM acting as a Domain Controller
- 1 Windows Client VM joined to the domain
- 1 Ubuntu Server VM running Docker services
- Internal users, groups and shared folders
- Basic backup and security documentation
