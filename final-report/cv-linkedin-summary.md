## Interview Explanation

This project is a practical lab where I built a small business-style IT environment from scratch.

I started by creating a Windows Server domain controller and configuring Active Directory. I created users, groups, and organizational units, then set up department file shares with permissions controlled through AD security groups.

I then installed a Windows 10 client and joined it to the domain to validate domain login and shared folder access.

After that, I added an Ubuntu Server to the lab, configured static networking, SSH access, and basic Linux hardening with UFW. I installed Docker and deployed an Nginx container hosting a custom internal GreenTech web page.

I also created a backup script for the web files and tested the restore process to confirm that the backup was usable. Finally, I documented logging, monitoring, security hardening, and incident response steps.

The project demonstrates practical skills across Windows Server, Active Directory, Linux, Docker, networking, access control, backup and restore, monitoring, and GitHub documentation.
